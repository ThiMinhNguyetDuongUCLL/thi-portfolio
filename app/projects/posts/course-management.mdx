A full-stack web application built to streamline course management and improve the learning experience for students, lecturers, and administrators.

This project delivers a robust platform for managing courses, student enrollment, and academic workflows. It features secure authentication, role-based access, and a responsive user interface designed for both desktop and mobile.

## Key Features

### Backend
- 8 REST controllers with a well-structured API design
- 25+ secured endpoints with role-based access control
- JWT authentication for secure sessions
- Multi-role support: Admin, Lecturer, and Student

### Frontend
- Responsive Next.js/React interface with 15+ reusable components
- Mobile-first design for cross-device compatibility
- 100% end-to-end test coverage using Cypress

### DevOps & Quality Assurance
- Automated CI/CD pipelines with GitHub Actions
- Database migrations managed with Flyway
- API documentation with Swagger UI
- Integration and E2E testing for long-term maintainability

## Technical Approach

The system was built with a clear separation between frontend and backend, following modern software engineering practices. Authentication and authorization ensure secure access for different user roles, while responsive design guarantees a smooth experience across devices.

## Tech Stack
- **Backend:** Java, Spring Boot, Spring Security, PostgreSQL/H2, Flyway, Maven, Swagger
- **Frontend:** Next.js, React, TypeScript, Tailwind CSS
- **Testing & DevOps:** Cypress, GitHub Actions
