A collaborative project with CCV Belgium to develop a chatbot widget for real-time customer interaction.

## Project Overview

**Duration:** October 2024 – January 2025

The chatbot was designed as a reusable widget to enhance user support and engagement. It supports role-specific responses, multiple languages, and voice interaction, all within a sleek, accessible UI.

## Key Features

### Core Functionality
- Real-time conversation handling with role-based responses
- Voice input integration using the Web Speech API
- Language localization for multilingual accessibility

### Frontend Development
- Built with React/Next.js and TypeScript
- Designed a responsive, minimizable interface

### Collaboration
- Worked in a 9-member Agile team with regular sprints and reviews
- Delivered a solution aligned with CCV Belgium's customer needs

## Technical Approach

The project focused on creating a seamless user experience through modern web technologies. The chatbot widget was designed to be easily integrable across different platforms while maintaining consistent functionality and accessibility standards.

## Tech Stack
- **Frontend & Features:** React, Next.js, TypeScript, Tailwind CSS, Web Speech API

## Impact

This project demonstrates real-world collaboration skills and client-focused development. Working with CCV Belgium provided valuable experience in enterprise software development, Agile methodologies, and delivering solutions that meet specific business requirements.
